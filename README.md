# Daconanalyzer
데이콘 심리 성향 예측 AI 경진대회

4만여명의 마키아벨리즘 테스트 답변과 국가 선거 투표 여부를 포함한 다양한 데이터들을
다양한 머신러닝, 딥러닝 알고리즘을 통해 학습시켜 국가 선거 투표자/미투표자의 심리학적 성향을 분석하고,
어떤 사람이 투표할 확률을 계산해보는 연구를 진행하였다.
